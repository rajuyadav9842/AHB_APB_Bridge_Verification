# AHB_APB_Bridge_Verification

This repository contains the RTL design and UVM-based verification of the AMBA AHB to APB Bridge using SystemVerilog. The project verifies communication between the AHB and APB protocols, including read/write transfers, bridge functionality, and protocol compliance.

## Features

* AHB to APB Bridge RTL Design
* UVM-Based Verification
* Read/Write Transactions
* Driver, Monitor, Scoreboard, Agent, Environment
* Functional Coverage
* Waveform Simulation

## Tools

* Verilog/SystemVerilog
* UVM
* QuestaSim
* Cadence Xcelium
